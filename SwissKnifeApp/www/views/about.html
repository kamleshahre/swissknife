Swiss Knife, v1.0

SwissKnifeApp is the solution for all your festival needs. Available on Android and iOS.

Developers

Swiss Knife is being brought to you by the following developers:

  • Nico Verbruggen (@nicoverbruggen)
  • Bjorn Van Acker (@bjornvanacker)

Technology

Swiss Knife makes use of the following technology:

  • Swiss Knife API (powered by Laravel)
  • Built for iOS and Android using Cordova
  • Main Javascript framework: Angular.js
  • Other libs: Bootstrap, FastClick.js, jQuery, FontAwesome, Modernizr, Leaflet

Supported devices

  • Builds for Android SDK version 18. Runs on all Android devices with minimum SDK support.
  • Builds for iOS 7.0 (with a few handmade fixes). Universal application runs on iPad and iPhone running iOS 7 or above.
